# paulocto-popper
Experiments for PaulOcto.

## Experiments

1. Edit your hosts:

    ```
    cd experiments
    vim conf/hosts/*
    ```

    Add files for client scaling experiments.

2. Edit your OSD disk:

   ```
   vim conf/osds.conf
   ```

3. Change the run script for each experiment:

   ```
   vim */run*.sh
   ```

3. Run your job:

   ```
   ./run.sh
   ```
